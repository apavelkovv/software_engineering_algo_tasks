## Описание задачи
Реализуйте функцию, которая определяет, является ли заданная скобочная последовательность правильной.

## Требования
- Написать функцию проверки корректности скобочной последовательности
- Функция принимает строку, содержащую только символы из набора: `()[]{}`
- Пустая строка считается правильной скобочной последовательностью
- Функция должна вернуть `True`, если последовательность правильная, иначе `False`

## Ограничения
- На вход подаётся одна строка без пробелов
- Алфавит строки: только символы `()[]{}`
- Ожидаемая асимптотика: время O(n), память O(n)

## Формат ввода
- Единственная строка со скобочной последовательностью

## Формат вывода
- Выведите `True` или `False`

## Примеры

```
Ввод:
{[()]}

Вывод:
True
```

```
Ввод:
()

Вывод:
True
```
