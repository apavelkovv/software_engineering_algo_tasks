## Описание задачи
Реализуйте функцию, которая печатает все элементы односвязного списка. 

## Требования
- Необходимо написать только функцию печати списка
- Функция принимает на вход голову списка
- Функция должна печатать каждый элемент списка на новой строке

## Ограничения
- Длина списка не превышает 5000 элементов
- Список всегда содержит хотя бы один элемент

## Формат ввода
- Параметр функции: голова односвязного списка

## Формат вывода
- Каждый элемент списка должен быть напечатан на отдельной строке
