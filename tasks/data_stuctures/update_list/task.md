## Описание задачи
Реализуйте функцию, которая удаляет элемент из односвязного списка по заданному индексу и возвращает голову обновлённого списка.

## Требования
- Необходимо написать только функцию удаления элемента
- Функция принимает на вход голову списка и индекс удаляемого элемента
- Функция должна вернуть голову обновлённого списка
- Нумерация элементов начинается с нуля

## Ограничения
- Длина списка не превышает 5000 элементов
- Список всегда содержит хотя бы один элемент


## Формат ввода
- Параметры функции:
  - node: голова односвязного списка
  - idx: индекс удаляемого элемента (нумерация с нуля)

## Формат вывода
- Функция должна вернуть голову обновлённого списка после удаления элемента


## Тесты
- Опишите от 5 тест кейсов, проверяющих корректность и алгоритмическую сложность предложенного решения

```python
class Node:
    def __init__(self, value, next_item=None):
        self.value = value
        self.next_item = next_item


def solution(node: Node, idx: int) -> Node:
    ...

# пример теста
def test():
    node3 = Node("node3", None)
    node2 = Node("node2", node3)
    node1 = Node("node1", node2)
    node0 = Node("node0", node1)
    new_head = solution(node0, 1)
    assert new_head is node0
    assert new_head.next_item is node2
    assert new_head.next_item.next_item is node3
    assert new_head.next_item.next_item.next_item is None
    # result is node0 -> node2 -> node3
```
