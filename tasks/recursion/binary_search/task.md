## Описание задачи
Реализуйте классический бинарный поиск по отсортированному по возрастанию массиву целых чисел. Функция должна вернуть индекс найденного элемента, либо −1, если элемента нет.

## Требования
- Входной массив отсортирован по возрастанию
- Индексация 0‑я
- Асимптотика: O(log n) по времени, O(1) по памяти

## Ограничения
- 0 ≤ n ≤ 10^6
- |a[i]| ≤ 10^9, |target| ≤ 10^9

## Формат ввода
- На вход функции подаются:
  - `arr: list[int]` — отсортированный массив
  - `target: int` — целевое значение

## Формат вывода
- Индекс элемента, равного `target`, либо −1, если такого элемента нет

## Примеры
```
Ввод:
1, 2, 2, 4, 7
2

Вывод:
1
```

```
Ввод:
1, 3, 5
2

Вывод:
-1
```
