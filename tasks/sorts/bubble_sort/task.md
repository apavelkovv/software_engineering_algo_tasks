## Описание задачи
Реализуйте алгоритм сортировки пузырьком для массива целых чисел по возрастанию. После каждого прохода по массиву, в котором были выполнены обмены, необходимо выводить текущее состояние массива.

## Требования
- Написать функцию сортировки пузырьком
- Сортировка по возрастанию
- После каждого «результативного» прохода (были обмены) фиксировать состояние массива
- Если массив изначально отсортирован, вывести его один раз

## Ограничения
- 2 ≤ n ≤ 1000 — длина массива
- Значения элементов: |a[i]| ≤ 1000
- Ожидаемая асимптотика: время O(n^2), память O(1) помимо вывода

## Формат ввода
- Первая строка: n — количество элементов массива
- Вторая строка: n целых чисел, разделённых пробелами

## Формат вывода
- Если за проход были обмены — вывести массив после прохода (одна строка, элементы через пробел)
- Если массив уже отсортирован к началу алгоритма — вывести массив один раз

## Примеры

```
Ввод:
8
3 7 9 4 3 1 8 5

Вывод:
3 7 4 3 1 8 5 9
3 4 3 1 7 5 8 9
3 3 1 4 5 7 8 9
3 1 3 4 5 7 8 9
1 3 3 4 5 7 8 9
```

```
Ввод:
5
1 2 3 4 5

Вывод:
1 2 3 4 5
```
