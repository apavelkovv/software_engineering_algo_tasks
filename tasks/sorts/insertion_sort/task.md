## Описание задачи
Реализуйте сортировку вставками для массива целых чисел по возрастанию.

## Требования
- Написать функцию сортировки вставками
- Сортировка по возрастанию
- Допускается использовать вспомогательные функции

## Ограничения
- 0 ≤ n ≤ 1000 — длина массива
- |a[i]| ≤ 1000
- Ожидаемая асимптотика: время O(n^2), память O(1)

## Формат ввода
- Параметр функции: `arr: list[int]`

## Формат вывода
- Функция должна вернуть новый отсортированный массив

## Примеры
```
Вход:
[9, 5, 1, 4, 3]

Выход:
[1, 3, 4, 5, 9]
```

```
Вход:
[]

Выход:
[]
```

- Опишите от 5 тест кейсов, проверяющих корректность и алгоритмическую сложность предложенного решения

```python
def insertion_sort(arr: list[int]) -> list[int]:
    ...


# пример теста
def test():
    assert insertion_sort([]) == []
    assert insertion_sort([3]) == [3]
    assert insertion_sort([9, 5, 1, 4, 3]) == [1, 3, 4, 5, 9]
    assert insertion_sort([5, 4, 3, 2, 1]) == [1, 2, 3, 4, 5]
    assert insertion_sort([2, -1, 2, -1, 0]) == [-1, -1, 0, 2, 2]
```
