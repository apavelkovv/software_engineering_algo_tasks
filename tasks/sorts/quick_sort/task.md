## Описание задачи
Реализуйте быструю сортировку для массива целых чисел по возрастанию.

## Требования
- Написать функцию быстрой сортировки
- Сортировка по возрастанию
- Допускается использовать вспомогательные функции

## Ограничения
- 0 ≤ n ≤ 10^5
- |a[i]| ≤ 10^9
- Асимптотика: средняя O(n log n), худшая O(n^2); память — глубина стека рекурсии O(log n) в среднем

## Формат ввода
- Параметр функции: `arr: list[int]`

## Формат вывода
- Функция должна вернуть новый отсортированный массив

## Примеры
```
Вход:
[3, 7, 9, 4, 3, 1, 8, 5]

Выход:
[1, 3, 3, 4, 5, 7, 8, 9]
```

```
Вход:
[]

Выход:
[]
```

def quick_sort(arr: list[int]) -> list[int]:

def test():
